Latest Projects

Mouse Tutor - Designed for software beginners

We propose the software tutorial that controls the mouse position as the cursor moves automatically on the software. The mouse position is controlled by XY table and the computer. This tutorial system has visual, auditory and physical information and is expected to improve the tutorial quality for users.

Edit Master - Designed for Drum Master

This is an web application that you can make taiko music scores. This system is very intersting because you can use YouTube music when you create taiko music scores. Also, this system uses Songle to calculate BPM and the starting time of music. It is very easy to make taiko music scores. You can share your taiko music scores with your friends and you can see your friends' taiko music scores.

Find out more

MLBMA - Designed for Nakano Library users

Meiji Library Books Management Application. With this app, you can check the state of books in Nakano Library. Some books are borrowed by someone, some books are not in the library, etc. Web scraping technology is used to get states of books. I used Laravel and Vue to make this.

Find out more

AR Taiko Editor - Designed for Drum Master

This system is for people who often use the simulator of Drum Master. You can make tja file physically. When you put AR markers in front of your web camera, you can see very cute taiko characters. This app recognizes AR markers and you can edit taiko score!

Find out more

Icon Rearrangement Method That Keeps Most Spatial Consistency - Designed for Interfaces

Many interfaces such as file explorers and windows setting screen use reflow. Reflow moves many icons and users are not able to use their spatial memory. We proposed the icon rearrangement method that keeps most spatial consistency.
